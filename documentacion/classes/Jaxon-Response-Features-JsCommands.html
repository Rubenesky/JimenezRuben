waitForCSS()

JsCommands.php : 429

Add a command to make Jaxon pause while the CSS files are loaded

public waitForCSS([int $iTimeout = 600 ]) : Response

The browser is not typically a multi-threading application, with regards to javascript code. Therefore, the CSS files included or removed with <Response->includeCSS> and <Response->removeCSS> respectively, will not be loaded or removed until the browser regains control from the script. This command returns control back to the browser and pauses the execution of the response until the CSS files, included previously, are loaded.

Parameters
$iTimeout : int = 600

The number of 1/10ths of a second to pause before timing out and continuing with the execution of the response commands

Return values
Response
